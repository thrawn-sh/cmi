# C.M.I.
python module to interact and read data from C.M.I. logger from [Technische Alternative](https://www.ta.co.at)

## Install depencies
```sh
$> poetry install
```

## Check source code
```sh
$> poetry run flake8
```

## Build package
```sh
$> poetry build
```

## Running scripts
```sh
$> poetry run cmi/...
```
